# Subscription Manager

A web application for managing recurring subscriptions and digital services such as Netflix, Spotify, ChatGPT Plus and Microsoft 365.

## Features

- Create, edit and delete subscriptions
- Manage predefined and custom digital services
- Track recurring monthly and yearly costs
- Import and export subscription data
- REST API with a Blazor web client

## Technology

- .NET 10
- ASP.NET Core Web API
- Blazor Server
- Entity Framework Core
- SQL Server
